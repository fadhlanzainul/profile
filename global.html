    DATA FLOW DIAGRAM (DFD)

    A graphical tool used to visualize the flow of data within a system, showing how data is processed and transferred between external entities, processes, and data stores. DFD consists of key elements such as processes, data flows, data stores, and external entities. This diagram helps in understanding how information flows through the system, as well as identifying inputs, outputs, and data management. DFD is used primarily in systems analysis to design or improve information systems.

    HTML

    HTML (Hypertext Markup Language) is the standard markup language used to create and design the structure of web pages. It provides the basic building blocks for web content, allowing you to format text, insert images, create links, and organize content using various elements and tags. HTML uses a system of tag to define different parts of a webpage, such as paragraphs, headings, links, images, tables, and other elements. It works alongside CSS (Cascading Style Sheets) for styling and JavaScript for interactivity, forming the foundation of modern web development.

    ENTITY RELATIONSHIP DIAGRAM (ERD)

    Entity-Relationship Diagram (ERD) is a high-level conceptual diagram that provides a visual representation of the structure of a database. Based on the concept of real-world entities and the relationships between them, ERDs help to identify the entities present within a system and define how these entities are related. As a user-friendly graphical tool, ERDs make it easier to model and visualize data structures, facilitating understanding for both technical and non-technical stakeholders. Widely used in database design, ERDs are considered a best practice for creating a blueprint of the data model before implementing the actual database, ensuring that the data structure is well-organized and efficient.

    CSS

    CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation and layout of web pages. It is used to define how HTML elements should be displayed on the screen, including aspects like colors, fonts, spacing, alignment, sizes, and other visual effects. CSS separates the content (HTML) from the design, making it easier to maintain and update the visual appearance of a website. By using CSS, you can apply consistent styles to multiple web pages, create responsive layouts, and enhance the user experience by making the website more attractive and functional. CSS works alongside HTML and JavaScript as one of the core technologies for building modern web applications.
